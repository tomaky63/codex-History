# TSMCとファウンドリモデルの成立——「設計と製造の分離」が生んだ台湾の先端半導体独占

## 俯瞰

1987年以前、半導体産業の常識は「チップを設計するなら自社で製造せよ」だった。IntelはCPUを設計しアリゾナ州の工場で製造した。TIもモトローラもそうだった。ファブ（製造設備）を持てない企業がチップを設計しても製造できないのだから、事業にならなかった。この常識を、テキサス・インスツルメンツ（TI）副社長を経て台湾政府系研究機関に招聘されていたモリス・チャンが崩した。

「設計は自分でやるが、製造は外部に委託する——そのための受託製造専業の会社」という前例のないモデルを掲げてTSMCを創業した。モリス・チャンの洞察の核心は「顧客と競合しない」という一点だ。自社製品を持たない純粋な製造専業者であれば、Intelも、Qualcommも、Appleも、TSMCの工場に最高機密の設計データを渡せる——互いに競合していないからだ。この信頼の構造が、ファブレス設計会社という新産業カテゴリーを可能にした。

1994年のTSMC上場と同時期にNvidia・Qualcomm・ARMなどのファブレス企業が台頭し、2009年にはAMDが自社製造部門をスピンオフ（GlobalFoundries）してファブレス転換した。2019年、TSMCの7nmプロセスがIntelの最先端ノードを上回り、世界の先端半導体製造の重心が完全に台湾に移った。この30年の軌跡は、「産業のバリューチェーンを分解する」ことで生まれる新しい競争優位の典型だ。

## 構造分析

### 何が起きていたか

**IDM時代（1960〜1987年）の産業構造**

半導体産業のIDM（Integrated Device Manufacturer）モデルは「設計・製造・販売の垂直統合」を前提とした。Intel、TI、モトローラ、NECなど先行企業は巨大な製造設備（ファブ）を自社保有し、自社設計チップを自社で製造した。このモデルの参入コストは極めて高かった——1980年代後半で1つのファブ建設に数億ドルが必要だった。結果として、チップ産業には設計能力と製造能力の両方を持てる大企業しか存在できなかった。

**TSMCの創業と初期顧客の獲得（1987〜1994年）**

モリス・チャンがTSMCを創業する際に設計した制約は逆説的だ——「製品を持たない会社」。これがビジネスモデルの核心だった。製品を持たないから、顧客の設計を受け取っても競合にならない。Altera（FPGA）、Xilinx（FPGA）などがいち早くTSMCを採用し、ファブを持たずに設計に専念できることを証明した。FPGAは設計複雑度が高くロジック密度が要求されるため、最先端プロセスへの需要が強く、TSMCの技術力向上のドライバーになった。

**ファブレス革命の加速（1994〜2009年）**

TSMC上場（1994年台湾証券取引所）は国際資本市場へのアクセスを開き、製造設備への投資規模を格段に引き上げた。この時期にNvidia（1993年創業、GPU設計に集中）・Qualcomm（CDMA/モバイルチップ）・Marvell（ネットワーク）・MediaTek（台湾系ファブレス）が台頭し、「ファブレス設計会社」という産業カテゴリーが実証された。AMDは2009年に製造部門をスピンオフしてGlobalFoundriesを設立し、AMD自体はファブレスに転換した。かつてIntelと互角に競ったIDM大手が製造を捨てたことは、IDMモデル後退の象徴だった。

**TSMCのプロセス技術優位の確立（2009〜2020年）**

2014年、AppleはiPhone 6のA8チップ（20nm）の製造をTSMCに委託し始めた——それ以前はSamsungが製造していた。世界で最も先進的なモバイルチップの製造を自社設計に依存しない「純粋製造専業者」に委ねた瞬間だ。同時期にIntelは10nmプロセスの量産化に繰り返し失敗し、2019年にTSMCが7nm（実効密度でIntelの10nmを上回る）を量産した。2020年にはAppleのM1（5nm）を筆頭に、AMD EPYC・Nvidia Ampere・Qualcomm Snapdragonなど世界の最先端チップをほぼ独占的に製造する地位を確立した。

### なぜそうなったか（力学）

**「顧客と競合しない」という信頼の構造的創出**

IDM時代、大手設計会社が最高機密の設計図を他社の工場に渡すことは自滅行為だった——製造技術と設計情報が漏洩すれば競合が有利になる。TSMCの「製品を持たない」という自己制約はこの問題を制度的に解消した。Appleがチップ設計の詳細をTSMCに渡しても、TSMCはAppleのスマートフォン競合にならない。Nvidiaが最先端GPU設計をTSMCに委託しても、TSMCはGPU市場に参入しない。この信頼が「それ以外では成立しない委託関係」を可能にした。

**規模の経済の非対称性**

IDMモデルでは各社が自社製品専用のファブを持つため、設備投資コストは自社売上で回収しなければならない。TSMCは複数の顧客の製造を集約するため、最先端の露光装置（1台数十〜数百億円のEUV装置）の投資コストを多数の顧客売上で分散できる。同等の技術水準を達成するためにIntelが必要な投資を、TSMCは顧客売上の集積によって正当化できる。顧客が増えるほどTSMCの投資余力が高まり、投資余力が高まるほどプロセス技術が向上し、技術が向上するほど顧客が増える——という正の循環が成立した。

**台湾政府のITRIによる技術的基盤の整備**

TSMCは真空から生まれたわけではない。1973年設立の工業技術研究院（ITRI）が1976年にRCA（米国）からCMOSプロセス技術をライセンス取得し、台湾に半導体製造の技術基盤を整備した。モリス・チャンは1985年にITRI院長として台湾に招聘され、1987年のTSMC創業はITRI技術の民間移転として実現した。日本の超LSI研究組合と同型の「政府主導の技術キャッチアップ」が起点だが、その後のビジネスモデルは全く異なる方向に進んだ。日本がDRAMの製造品質向上に集中した一方、台湾は「製造を武器に設計分離を促進する」という産業構造変革を起動した。

**ムーアの法則維持コストの増大と「選択と集中」の圧力**

半導体製造の微細化（プロセス技術の向上）は指数関数的にコストが増大する。1990年代のファブ建設コストが数億ドルだったのに対し、2020年代の最先端ファブ（2nm以降）は数兆円規模だ。この投資規模は、単独製品で回収しようとするIDMには限界を生む。AppleはA-seriesチップのみ、IntelはPCプロセッサのみで回収しなければならないのに対し、TSMCは世界中の先端チップ需要を全て集約して回収できる。微細化が進むほど「集中型製造者」の経済合理性が高まる構造だ。

### 構造図

```
【IDM時代（〜1987年）】

 [Intel / TI / NEC 等]
  ┌── 設計部門（CPU・メモリ設計）
  ├── 製造部門（自社ファブ）
  └── 販売部門
  ※参入条件：設計能力 × 製造能力の両方が必要
  ※新興チップ会社はどちらかが欠けると成立しない

【ファブレス+ファウンドリ時代（1987年〜）】

 [ARMなど                 [Qualcomm/Nvidia/Apple Siliconなど]
  IPコア設計] ──→         設計のみ（ファブレス）
                                  ↓ 設計データを委託
              ┌────────────────────────────┐
              │  TSMC（純粋受託製造 = ファウンドリ） │
              │  ・製品を持たない → 顧客と競合しない  │
              │  ・複数顧客の製造集約 → 最大規模の投資│
              │  ・EUV露光装置等の先端設備を独占保有  │
              └────────────────────────────┘
                                  ↓ 製造済みウェハ
              [後工程業者（ASE/Amkorなど）] → 最終製品

  ・ファブレス会社：設計に集中 → Nvidia・Qualcomm等が誕生可能に
  ・TSMC：全顧客の投資を集約 → Intel一社の投資規模を超えていく
```

## 転換点の意味

### TP1: 1994年 TSMC上場とファブレス産業の確立——ビジネスモデルの実証

1994年のTSMC台湾証券取引所への上場は、「製造専業ファウンドリ」が投資家に評価される事業として成立することの証明だ。上場によって資本市場への接続が生まれ、製造設備投資の規模が次のレベルに引き上げられた。同時期に、TSMCの存在を前提として設計に集中するファブレス企業の第一世代（Altera, Xilinx, Marvell等）が確立したことで、「ファブレス+ファウンドリ」という産業構造のデュアルが完成した。

### TP2: 2009年 AMD/GlobalFoundries設立——IDMモデルの自己否定

AMD（Advanced Micro Devices）はIntelと並ぶCPUのIDM大手だった。そのAMDが製造部門をスピンオフして独立会社（GlobalFoundries）を設立し、設計部門のみのファブレスに転換した。これは「IDMモデルの否定」を当事者が行ったという意味で歴史的だ。IDMモデルを続けることが財務的に困難になり、製造コストを分離して設計に集中せざるを得なくなった。この転換がTSMCへのAMDのCPU委託につながり、AMDのEPYC（サーバーCPU）がTSMCの7nmで大成功を収めることになる。

### TP3: 2019年 TSMC 7nm——Intelを逆転した技術的臨界点

Intelは1960年代から半導体プロセス技術のリーダーだった。「インテル入ってる」という信仰は製造技術の優位性に支えられていた。2019年、IntelはPC向けCPUで10nmプロセスへの移行を繰り返し遅らせた一方、TSMCはAppleのA12・AMDのZen2で7nmを量産した。「ファウンドリの受託製造業者がIDMの製造技術の覇者を逆転した」瞬間だ。この逆転は一時的なものではなく、その後TSMCは5nm（2020年）・3nm（2022年）と先行し続けた。製造技術の主導権が台湾に恒久的に移ったことを宣言する転換点。

## 反対と執念

モリス・チャンが1987年に「純粋受託製造専業」というモデルを提唱した時、同業者・投資家の多くは懐疑的だった。「製品のない製造会社が、製品のある会社の先端技術に追いつけるはずがない」という論理だ。製品ロードマップがなければ、どのプロセス技術に投資すべきか分からない——これがIDMモデルの通念だった。モリス・チャンの反論は「複数の顧客の多様な技術要求に対応することが、むしろプロセス技術の幅と深さを広げる」というものだった。

TSMC創業初期、最先端プロセス技術ではIntel・IBMに大きく後れを取っていた。顧客が委託できる製品は、最先端を必要としないFPGAや通信チップが中心だった。モリス・チャンは「最先端顧客を引き込むには技術が必要で、技術向上には最先端顧客の注文が必要」という鶏と卵を、利益の全てを研究開発に再投資することで解決した。1990年代を通じてTSMCの設備投資・R&D比率は売上高の20〜30%に達し、これが技術追い付きの原資となった。

## 考察・私見

TSMCの事例で最も深く考えさせられるのは「制約がビジネスモデルを作る」という逆説だ。「製品を持たない」という自己制約は一見、競争力を弱める。しかし実際はその制約が「顧客の信頼」という、金で買えない競争優位を生んだ。AppleがチップをSamsungからTSMCに移したのは、SamsungがAppleのスマートフォン競合であるという「利益相反」を解消するためだった。制約が差別化になった典型例だ。

地政学的含意は産業史の文脈を超える。TSMCが台湾に存在し、台湾以外では5年以内に複製不可能な製造能力を持つという事実が、台湾海峡の緊張下で「台湾を失うコスト」を無限大に近づけた。これは意図的な地政学戦略ではなく、純粋な競争優位の追求の結果として生まれた「シリコンシールド（Silicon Shield）」だ。産業の競争優位が国家安全保障と等価になるというこの構造は、20世紀の石油産業における「中東の地政学的重要性」と同型だ。OPECが石油を武器にしたように、TSMCは（意図せずして）先端半導体製造を地政学的カードにした。

日本半導体産業との対比は鮮明だ。日本（NEC・東芝）はDRAM製造品質で世界首位に立ったが、「ファブレス＋ファウンドリ」という産業構造の分離に乗り遅れた。TSMCは同じ時代に「どのゲームを戦うか」を正しく選択した——DRAM製造の品質ゲームではなく、製造専業プラットフォームの信頼ゲームを戦った。日本が誰でも製造できる汎用メモリの製造コスト競争に留まった一方、TSMCは「製造しか出来ないが、その製造を外すと世界中のチップが作れなくなる」という代替不可能性を構築した。「何を作るか」より「誰のために作るか」の問いへの答えが、産業の位置を決めた。
