# Costcoの会員制モデルと利益の源泉の逆転

## 俯瞰（Overview）

普通の小売業は「商品を仕入れて、マークアップして、利益を出す」という構造だ。Costcoはこれを根本から否定した。商品はほぼ原価で売る。利益は年会費から稼ぐ——これがCostcoの構造だ。

2022年度を例にとると、Costcoの総売上は約2300億ドルだが、商品の粗利はわずか11%台。対してウォルマートは約25%、ターゲットは約30%。Costcoの商品利益は「設備・人件費・物流費をほぼカバーするだけ」の水準に設定されている。代わりに会員費は44億ドルで、これがほぼそのまま営業利益になる。

この構造が成立する理由は「年会費を払った顧客の行動変化」にある。年会費（現在$65〜$130）を払ったCostcoの会員は「元を取ろう」という動機で来店頻度・購入量が増え、他の小売店より圧倒的に多く買う。その購買力がCostcoのバイイングパワー（仕入れ力）を支え、安い仕入れ価格が安い販売価格を可能にし、さらに会員を増やす——この正のフィードバックがCostcoの競争力の本質だ。

## 構造分析

### 何が起きていたか

**1976-1983：原型「Price Club」と倉庫型小売の発明**

ウェアハウスクラブ（倉庫型会員制小売）の最初の形態は、ソル・プライスが1976年にサンディエゴに開いた「Price Club」だった。当初は中小企業向けの業務用卸売として始まり、後に一般会員向けに拡大した。Price Clubの基本思想は「品揃えを絞り、大量に仕入れ、マークアップを最小にすることで、会員に本物の値引きを提供する」だった。

ソル・プライスの弟子の一人がジム・シネガルだった。シネガルはPrice Clubで学んだ後、1983年にジェフ・ブロットマンとともにシアトルでCostcoを設立した。1993年にPrice ClimbとCostcoが合併してPriceCostcoとなり、1997年にCostcoと改称した。

**1983-1993：Costcoの初期展開と「利益の源泉の明確化」**

Costcoが当初から他の小売業と異なっていたのは「商品粗利の上限」の設定だった。創業者のジム・シネガルは「生鮮食品以外の全製品の粗利は14%を超えない」というルールを設けた。後にこれは「ブランド品は14%、コーポレートブランド（カークランドシグネチャー）は15%」という具体的な上限になった。

なぜこのルールが重要か——「粗利上限」を社内ルールとして明示すると、バイヤーが「この商品は20%取れるから無理に入れる」という歪んだインセンティブが消える。常に「最安値で提供できるか」が判断基準になる。シネガルは「我々は顧客に対して最大の価値を提供することを最優先にする」という哲学を、実際の数字のルールとして実装した。

**1994-2010：カークランドシグネチャーとバイイングパワーの確立**

1995年頃に本格展開が始まったCostcoのプライベートブランド「Kirkland Signature（カークランドシグネチャー）」は、Costcoのビジネスモデルの洗練を示す。カークランドの基本思想は「ナショナルブランドより品質が同等以上で、価格は20%安い」だ。カークランドの品質が高い理由は、Costcoが「同等品のナショナルブランドのメーカーに製造委託する」ことが多いからだ——例えばKirkland Signatureのウイスキーの一部は有名蒸留所製、Kirkland Signatureのオリーブオイルは高品質イタリア産だという話が広く知られている（一部は確認可能、一部はコミュニティの推測）。

このアプローチで、Costcoは「ナショナルブランドとのバイイングパワー交渉」においても強い立場を保てる。「うちはKirklandで置き換えられる。それでも納得できる卸価格を出せるか」という交渉力だ。

**2005-2020：Amazon Primeとの比較——「会員制モデル」の普及**

AmazonがPrime（2005年開始）を年会費制サービスとして展開した時、Costcoの影響は明確だった。ジェフ・ベゾスは「顧客との関係をサブスクリプションで強化する」というアイデアのヒントとして、Costcoの会員モデルを参照していたと知られている。2024年時点でAmazon Primeは2億人超の会員を持つが、Costcoの会員（1.3億人超）もその後を追う。両者の本質的な違いは「Costcoは物理的な倉庫店に来ることで価値を享受する」「Primeはオンラインで価値を享受する」という点だが、「年会費を払うことで顧客ロイヤルティと購買頻度を高める」という構造は同一だ。

### なぜそうなったか（力学）

**「元を取ろう」の心理——サンクコストを味方にする**

年会費を支払った顧客は「せっかく払ったから元を取らなければ」というサンクコスト効果（本来は合理的でないが）が働き、来店頻度と購買量が増える。これは合理的な判断ではなく心理的なバイアスだが、Costcoにとっては来店頻度と購買量の安定的な増加をもたらす。加えて「会員でしか入れない」という排他性が「入っていること自体のステータス」を生む。

**品揃えの絞り込みとSKU数の少なさ**

普通のスーパーマーケットは3万〜5万SKU（品番）を持つ。CostcoのSKU数は約4000。これは「同じカテゴリで最も売れる1〜2品種しか置かない」という選択だ。品揃えが少ないと一見顧客に不便だが、バイヤーにとっては「同一品を大量に仕入れることで交渉力が最大化できる」。シアトルのCostcoがケチャップを1種類しか置かないなら、その1種類を全米・全世界のCostcoに供給するメーカーは「Costcoに選ばれること」のために限界まで価格を下げる。

**従業員への高待遇というサプライサイドの設計**

CostcoはWalmart・Targetと比べて従業員の時給が高く（2023年時点で最低$17.50/時、平均は$25超）、医療保険カバレッジも手厚い。これは一見「コストを上げている」ように見えるが、離職率の低下（=採用・訓練コストの削減）と従業員の高いモチベーション（=顧客体験の質の維持）につながっている。小売業の離職率が年100%を超えることが珍しくない中、Costcoの離職率は5〜6%台（正社員）という異常値を保つ。

**「ガスリン（ガソリンスタンド）・タイヤセンター・フードコート」というロスリーダーの設計**

Costcoの店舗内フードコートの象徴は「$1.5のホットドッグ＋ドリンクセット」だ。1985年の導入以来40年近く価格が据え置かれているこの商品はほぼ間違いなく赤字だが、Costcoはこの価格を変えることを拒否し続けている（2023年、シネガルはナラエンCEOに「値上げしたら殺す」とジョークで釘を刺したというエピソードが有名）。ガソリンスタンドも市場最安値水準に設定され、「Costcoに行ったついでに給油する」という習慣を作ることで来店頻度を高める。これらロスリーダーの存在が「Costcoは得する場所」という知覚を強化し、会員の更新率を支えている。

### 構造図

```
[Costco会員（$65/年〜）]
    ↓ 「元を取ろう」来店頻度増・大量購入
    → 年会費 $65-130 × 1.3億会員 = 会員費収入 $44億（≒ 営業利益の大半）
    
[Costcoオペレーション]
    商品粗利率: 11〜12%（業界最低水準）
    粗利上限ルール: ブランド品14%、PB品15%の上限を社内ルール化
    SKU数: 約4000（一般スーパーの1/10）
    → 絞り込み×大量発注 → 仕入れ価格の最大化

    ↑ バイイングパワーの好循環
[メーカー・仕入れ先]
    「Costcoに選ばれる＝全米・全世界に一品種で大量供給できる」
    → 通常の卸価格より大幅に安い価格でCostcoに提供せざるを得ない

[競合スーパー（Walmart・Target・Kroger等）]
    粗利率25〜30%で運営 → 商品価格でCostcoに勝てない
    会員制でないため来店頻度の心理的強制なし

フードコート($1.5ホットドッグ)・ガソリンスタンド（最安値）
    = ロスリーダー → 「Costcoは得する場所」の知覚強化 → 会員更新率93%
```

## 転換点の意味

### TP1: 1976年 Price Clubの設立——「会員制倉庫型小売」の発明

ソル・プライスが1976年にサンディエゴで開いたPrice Clubは、当初は中小企業向けの業務用卸としてスタートした。「一定額の年会費を払った会員だけが入れる倉庫型店舗」というフォーマットの最初の完成形だった。

なぜ転換点か: 「会員資格の対価として商品を安く提供する」という利益構造の逆転がここで発明された。従来の小売業は「誰でも入れる代わりにマークアップで稼ぐ」。Price Clubは「入店に年会費を払う代わりに、ほぼ原価で買える」という交換を提示した。この交換の論理が、後のCostco・Amazon Prime・Netflix等の「会員制（サブスク）による顧客ロイヤルティ設計」すべての原型になった。

### TP2: 1983年 Costco設立・「粗利上限14%ルール」の制度化

ジム・シネガルが設立したCostcoが最初から持っていた「商品粗利を14%以下に抑える」というルールは、Price Clubの思想を組織的規律として実装したものだった。

なぜ転換点か: 多くの小売業で「バイヤーが仕入れ価格と棚割りの裁量を持つ」ことでインセンティブが歪む——利幅の良い商品を棚に入れたくなる。Costcoはこの裁量を数字のルールで奪った。「何%まで稼いでいいか」の上限を社内で明文化することで、「顧客への価値提供の最大化」と「利益の安定」が同時に達成される構造を作った。これはコーポレートガバナンスとビジネスモデル設計が一体になった珍しい事例だ。

### TP3: 2020年代 Amazon Primeとの競合——「会員制モデル」の主戦場化

Amazon Primeが2億人超の会員を持つ巨大なプラットフォームになった2020年代、Costcoは「物理的な体験」という独自の価値で会員更新率93%を維持している。Amazonがオンラインで何でも届けられる時代でも、「Costcoに行くこと自体がエンターテインメント」という体験価値——大型商品・試食・フードコート——が差別化要因になっている。

なぜ転換点か: デジタルとフィジカルの会員制モデルが直接競合する時代が到来した中で、Costcoが「物理的な倉庫体験」を武器に高い更新率を維持していることは、「会員制の価値は利便性だけでなく体験に依存する」という論点を提示している。Costcoの会員制とAmazon Primeの会員制は、表面上は同じ「年会費制サブスク」だが、顧客に提供している価値の本質が異なる。

## 反対と執念

ジム・シネガルが「$1.5のホットドッグセット」の値上げを拒否し続けた話は、Costcoの企業文化の象徴として語り継がれている。2009年に食材費の高騰でCFOがホットドッグの値上げを提案した際、シネガルは「値上げするなら君を殺す。理解できるか」と言ったとされる（冗談だが本質を突いた言葉）。同年、Costcoはホットドッグ工場を内製化して原価を下げ、価格を維持した。

ウォール街からの圧力も継続的にあった。「粗利率を少し上げるだけで利益が大幅に改善する」というアナリストの主張に対し、シネガルは「それは間違った方向だ。私たちは長期的な顧客ロイヤルティに投資している」と繰り返し答えた。2012年に引退するまで、シネガルのCEO年俸は約350万ドルと同規模の企業のCEOより低く抑えられており、「従業員と顧客への公正さ」を体現した経営スタイルだった。

## 考察・私見

Costcoのモデルが最も示唆深い点は「利益の源泉を商品から会員資格に移動させる」というビジネスモデルの設計だ。商品の価格競争力を最大化するために、利益の源泉を別の場所（年会費）に置く——この「利益分離設計」は、消費者への最大の価値提供と企業の持続的利益成長を両立させる希少な構造だ。

AmazonがCostcoを参照してPrimeを作ったという事実は重要だ。デジタルの世界では「年会費 + 手厚い特典で顧客ロイヤルティを確保する」というモデルがNetflix・Spotify・Appleのエコシステムに広がった。Costcoは1983年にこの原理を物理的な小売業で実証した先駆者だった。

一方で、Costcoのモデルには重要な制約がある——「物理的な大量購入が合理的な顧客セグメント（世帯・中小企業）にしか機能しない」という点だ。一人暮らしや高齢者には「大容量商品を安く買う」価値が薄い。Costcoが都市部より郊外に多い理由はこの顧客セグメントの制約を反映している。

## 未解決の問い

- 粗利上限14%ルールはどのように運用されているのか——バイヤーへのインセンティブ設計（ボーナス指標は何か）と、このルールが実際に守られているかの組織的保証は？
- カークランドシグネチャーのOEMメーカーの実態——どのメーカーが製造しているか非公開だが、この「秘密主義」はコカ・コーラの秘密レシピと同様に、ブランドへの神話化として機能しているか
- Amazon Primeとの競合が激化する中で、Costcoが「物理的体験」の価値を維持できる地理的・人口統計的な条件は何か
- Costcoが会員制を日本・アジアで展開する際に直面した文化的・商慣習的な障壁——日本のCostcoが成功した一方で、「大量購入文化」のない文化圏での限界は？
- 2030年以降、配達サービスとのハイブリッド（InstacartとのCostco提携等）はフィジカル体験の価値を代替するか、それとも補完するか
