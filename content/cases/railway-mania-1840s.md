# 鉄道バブルと議会承認制度——1840年代イギリスで9500マイルのインフラを副産物として生んだ投機熱

## 俯瞰

1840年代のイギリスで起きた「鉄道バブル（Railway Mania）」は、歴史上もっとも研究された投機バブルのひとつでありながら、もっとも誤解されやすい事例でもある。通常の理解では「狂気の投機が多くの人を破産させた」で終わるが、この事例の本質は「崩壊した投機が9500マイルの鉄道インフラという巨大な遺産を残した」点にある。

事件の外形はシンプルだ。1844年にGladstone鉄道法が成立し、政府が一定条件で鉄道会社を買い取る権利（20年後に保証収益の1.5倍で）を明記したことで、投資家は「政府が最終的に買い取るなら安全だ」と解釈した。1845年には272件もの鉄道会社設立の私立法案が議会に提出され、全国の土地・農場・教会基金まで株式投資に向かった。株価は1845〜46年の2年で数倍に膨れ、1847年の金融恐慌で崩壊した。

しかしバブルの崩壊と実際のインフラ建設は別の話だ。1836〜1850年に承認された路線の大部分は実際に建設され、英国には20世紀まで使われ続ける鉄道網が残った。「誰かが損をしたことで、後世全員が得をした」——これが鉄道バブルの構造的教訓であり、通信バブル（2000年代）で光ファイバーが残ったのと同型のパターンである。

## 構造分析

### 何が起きていたか

**第一段階（1825〜1843）: 蒸気鉄道の実証と第一次小バブル**

1825年のストックトン・ダーリントン鉄道、1830年のリバプール・マンチェスター鉄道が、蒸気鉄道が馬車に代わる商業的輸送手段として機能することを実証した。1836〜37年には最初の投機的過熱が起きたが、資本が限られておりすぐに収束した。

**第二段階（1844〜1846）: 本格バブルの形成**

Gladstone鉄道法（1844年）は第三等運賃規制（1ペニー/マイル）と政府買取権を定めたが、市場はこれを「政府の鉄道保証」と読み取った。1845年11月に提出された鉄道法案は272件（前年比4倍以上）に達し、要求された資本総額は£2億（現代価値で約£200億）を超えた。法案提出には議会費用として数万ポンドが必要で、法案代理業者（Parliamentary agents）が急増した。

George Hudsonは1845年時点でヨーク・ノース・ミッドランド鉄道など複数路線を支配し、「Railway King」と呼ばれた。彼は既存株主への高配当維持のために新規発行株の払込金を配当に充てるという、後にポンジー・スキームと認定される手法を使っていた。しかし株価が上昇し続ける間は誰も異を唱えなかった。

**第三段階（1847）: 崩壊のトリガー連鎖**

1847年春、複数の要因が重なった。アイルランドのジャガイモ飢饉（1845〜47年）による穀物価格高騰で農業向け資金需要が急増。イングランド銀行が1847年8〜9月に割引率を3%から6%へ急引き上げ。建設中の路線で資本請求（calls）が相次ぎ、既存株主が追加払込を迫られた。鉄道株は6ヶ月で平均70〜80%下落し、数千人の中産階級投資家が全財産を失った。

**第四段階（1848〜1855）: 統合と遺産確定**

バブル後、資金不足で建設途中の路線は大手鉄道会社に吸収された。ミッドランド鉄道、ロンドン・ノース・ウェスタン鉄道など後世に続く幹線鉄道会社がこの時期に形成された。1849年にGeorge Hudsonの不正が発覚し失墜したが、彼が推進した路線の大部分は完成していた。

### なぜそうなったか（力学）

**議会承認というボトルネックが競争的投機を起動した**

英国では鉄道会社の設立には個別の議会私立法（Private Act of Parliament）が必要だった。これは「承認される数に上限がある」という希少性を作り出し、「今申請しないと競合が枠を取る」という心理的圧力を生んだ。あたかも土地収用のような「早いもの勝ち」の構造が、資本の急速な動員を促した。

**Gladstone法の誤読が投資を正当化した**

政府が20年後に買取権を持つということは「政府が損失を補填する」ではなく「収益性が高い路線は政府が買い取る可能性がある」という意味だった。しかし投資家はこれを「政府保証付き」と解釈した。この誤読が「鉄道への投資はリスクが低い」という集団的信念を形成し、本来リスクに慎重であるべき中産階級まで参入させた。

**情報の非対称性と「Railway King」のモラルハザード**

George Hudsonのような内部者は自分が支配する会社の財務状況を知りながら、株式を高値で売却し続けた。外部の投資家は配当実績と株価上昇しか見えず、その配当が新規株主からの払込金で賄われていることを知る術がなかった。バブル期に成功者に見えた人物ほど、その成功の多くが詐欺的手法に基づいていた。

**「崩壊しても鉄道は残る」という誤った安全感**

「たとえ会社が潰れても鉄道路線という実物資産が残る」という論理で投資が正当化された。実際にはコストが高く採算の取れない路線は誰も引き継がず、建設途中で放棄された。実物資産の残存は経営の生き残りを保証しない。

### 構造図

```
【投機の起動構造】
  Gladstone鉄道法（1844）
  ├─「政府が20年後に買取権」→ 投資家が「政府保証」と誤読
  └─「第三等運賃規制」→ 鉄道の社会インフラ的性格を政府が認定

  議会承認というボトルネック
  ├─「枠に限りがある」→ 競争的早期申請を促進
  └─「申請費用が数万ポンド」→ 真剣な投資家と投機家が混在

【バブル形成の正のフィードバック】
  株価上昇 → 新規参入者増加 → 配当原資を新株払込で確保（Hudson手法）
            → さらに株価上昇 → 中産階級・農民まで参入
            → 1845年11月：272件の新規申請

【崩壊のトリガー連鎖（1847）】
  穀物価格高騰（アイルランド飢饉）
  + BoE利上げ（3%→6%）
  + 建設中路線の追加払込請求（calls）
  ↓
  鉄道株70-80%下落 → 中産階級の破産 → Hudson不正発覚（1849）

【崩壊後の遺産】
  建設された路線（9500マイル）
  └─ 大手鉄道会社が吸収（ミッドランド、L&NW等）
     └─ 19世紀後半の英国経済基盤を形成
```

## 転換点の意味

### TP1: 1844年 Gladstone鉄道法——投機を正当化する規制の誕生

なぜ転換点か: Gladstone法以前の鉄道投資は純粋に採算性で判断されていた。同法の成立は「鉄道は社会インフラであり国家が関与する」という信念を制度化し、投資リスクを下げるものとして市場に解釈された。「規制による正当化」がリスク認知を歪め、本来は慎重になるべき投資家層を引き込んだ。規制は市場を抑制するためではなく、意図せずして投機を解放するトリガーになりうることを示した最初の大規模事例。

### TP2: 1845年——全階層参加の投機絶頂と「バブルが成立している」という集団的確認

なぜ転換点か: 1845年に農民・聖職者・家政婦まで鉄道株を購入し始めた時点で、バブルは「賢い人の投資」から「全員参加の賭け」に転化した。これはバブル研究でいう「最後の買い手」の出現であり、価格を支える新規参入者が枯渇し始めたことを意味した。同時に272件の新規申請は、承認ゲートの希少性という投機起動要因がもはや機能しなくなったことも示す（誰でも申請できるなら希少でない）。転換点は「過熱の頂点」であり「崩壊の起点」だった。

### TP3: 1847年 イングランド銀行の利上げと金融恐慌——「現実」の逆襲

なぜ転換点か: 鉄道バブルの崩壊は鉄道自体の問題ではなく、外部の金融環境の変化によってトリガーされた。農業危機（穀物価格高騰）が資本を鉄道から農業に吸収し、BoEが利上げで追い打ちをかけた。これは「バブルは内部矛盾だけでは崩壊しない。外部のショックが引き金を引く」というメカニズムを示している。また、建設中路線への追加払込請求という「実物経済のリアリティ」が投機家を直撃したことも重要で、「紙の上の価格」と「実際に必要な現金」の乖離が臨界を超えた瞬間だった。

## 考察・私見

この事例で最も興味深いのは「バブルの崩壊と遺産の形成が分離している」という構造だ。投資家は損をしたが、9500マイルの鉄道は残った。これはバブルを「失敗」と評価するのか「成功」と評価するのかを誰の視点で見るかによって決まる、という問いを突きつける。

現代の類比は明白だ。2000年代の通信バブルで過剰敷設された光ファイバーが、2010年代のインターネット経済の基盤になった。電気自動車充電インフラへの過剰投資も同様のパターンをたどるかもしれない。「今の投資が10〜20年後に別の誰かに回収される」という構造は、インフラ産業における投機の特殊なメカニズムとして繰り返し現れる。

George Hudsonの失墜は、バブル期に「成功者」と見なされた人物の多くが実力ではなく詐欺に依存していたことを示す。株価が上がり続ける間は詐欺が成功に見える。これはバブルの「隠蔽効果」であり、崩壊して初めて誰が水着を着ずに泳いでいたかが分かる（バフェットの表現を先取りした事例ともいえる）。

もうひとつ考えたいのは「議会承認」という制度的ボトルネックの役割だ。これは投機を抑制するための仕組みだったが、実際には「希少なゲートに殺到する」という心理を起動して投機を加速した。制度設計者の意図と市場参加者の反応が逆になる事例——規制が意図せず投機のアクセルになった。

## 未解決の問い

- Gladstone自身は市場がどう反応するかを予測していたか？議会での法案成立の経緯と議論の内容を調べたい
- 9500マイルのうち、採算が取れた路線と取れなかった路線の比率はどのくらいか？「役に立った遺産」と「使われない遺産」の割合
- 投資家の社会属性（職業・資産規模）と損失規模の分布——「中産階級の破産」はどの程度の規模・深刻さだったか
- George Hudsonが刑事訴追されなかった理由——同時代の法制度ではポンジー的手法が違法でなかったのか
