# コカ・コーラのボトラー制度と濃縮液ビジネスの発明

## 俯瞰（Overview）

「コカ・コーラはどこで儲けているのか」——答えは「ボトルの中身ではなく、濃縮液を売ることで」だ。コカ・コーラ本社は実はコーラを作っていない。コーラを作るのはボトラー（充填業者）で、コカ・コーラ本社はボトラーに濃縮液（コンセントレート）を売り、そのレシピと「コカ・コーラ」ブランドの使用権を管理しているだけだ。

この構造は偶然の産物だった。1899年、アトランタの弁護士2人がコカ・コーラの所有者アサ・キャンドラーに会い、「瓶入りコーラの全米販売権をください」と頼んだ。キャンドラーは「瓶入りは当たらない」と思っていたため、1ドルで全米の永久ボトリング権を渡してしまった。この「1ドル契約」が産業史に残る出来事になる。

ボトラー制度の本質は「何を保持し、何を委ねるか」の設計にある。コカ・コーラが保持したのは2つだけ——秘密の製法（コンセントレートの配合）とブランド。それ以外のすべて、工場・配送車・冷蔵庫・営業チームはボトラーが持つ。コカ・コーラは製造設備にほぼ投資せずに世界中に展開できた。一方でボトラーは「独占テリトリー」という強力な経済的インセンティブを与えられ、地域市場の開拓に命をかけた。

この構造は100年以上機能しているが、何度か深刻な危機を迎えた。中でも最大の問題は「原液価格の固定」だった。1899年の契約はシロップの価格を固定していたため、第一次大戦後の砂糖価格高騰でコカ・コーラは赤字になる可能性があった。訴訟で一部解決したが、根本的な再設計は1980年代まで待たねばならなかった。

## 構造分析

### 何が起きていたか

**ボトラー制度の構造**:
コカ・コーラのバリューチェーンは4層に分かれている。
1. **コカ・コーラ本社**: 秘密の濃縮液を製造し、ボトラーに販売。ブランド・広告・製品開発を管理。
2. **ボトラー（充填業者）**: 濃縮液を仕入れ、水・炭酸・砂糖を加えてコーラを完成させ、瓶/缶に詰めて配送。独占テリトリー制。
3. **流通チャネル**: スーパー、コンビニ、レストラン、自販機。
4. **最終消費者**.

ボトラーは独占テリトリーを持つため、そのエリアではコカ・コーラのボトラーしか正規製品を販売できない。この独占権がボトラーにとっての巨大なインセンティブであり、事業への投資を正当化する根拠だった。

**「1ドル契約」とその後の問題**:
1899年の契約は永久固定価格でシロップを供給するという内容だった。ベンジャミン・トーマスらボトラーにとっては夢のような条件。しかし第一次大戦で砂糖価格が高騰すると、コカ・コーラは大赤字リスクに直面した。社は価格引き上げを試みたが、ボトラーは契約違反だと提訴。法廷闘争は1920年代まで続き、最終的に「一部の価格改定は可能」との和解が成立した。しかし根本的な契約構造は変わらず、ボトラーの交渉力は依然として強かった。

**ゴイズエタ革命（1980年代）**:
1981年にCEOに就任したロベルト・ゴイズエタは、コカ・コーラを「濃縮液カンパニー」と再定義した。製造・充填・配送は資本集約的でリターンが低い。コカ・コーラの価値の源泉はブランドと秘密の製法だけだ——この論理でボトラー事業を整理し始めた。1986年には経営が困難になった中小ボトラーを束ねて「コカ・コーラ・エンタープライズ（CCE）」を設立し、その49%を保持しながら上場させた。これにより、コカ・コーラはボトラーを実質的にコントロールしながらも、バランスシートから外すことに成功した。

### なぜそうなったか（力学）

**「秘密の製法」という非合理的な堀**:
コカ・コーラの競争優位の核心は、化学的にはほぼ解析可能なコーラの配合を「秘密」として神話化したことだ。「メキシコ銀行の金庫に保管されている」という伝説的な話は、特許で保護できる技術より強力な心理的参入障壁を生んだ。ボトラーは製法を知らないため、原液はコカ・コーラからしか買えない。これが交渉力の源泉。

**独占テリトリーとフランチャイジーの投資インセンティブ**:
ボトラーに独占テリトリーを与えることで、ボトラーは工場・配送システム・営業チームに積極投資した。もし複数のボトラーが競合するテリトリーなら、誰もインフラに投資しない（フリーライダー問題）。コカ・コーラはこのインセンティブ設計で、自己資本ゼロで全米をカバーする流通網を構築した。

**ブランド広告費の非対称負担**:
コカ・コーラ本社が全国ブランド広告を担い、ボトラーが地域プロモーションを担う分業が成立した。全国広告は「コカ・コーラを飲みたい」という需要を創出し、ボトラーはその需要を地域で刈り取る構造。本社の広告投資がボトラー全員の売上を押し上げるため、ネットワーク外部性が働く。

**コンセントレートのマージン vs ボトリングのマージン**:
コカ・コーラの収益性の秘密は、濃縮液販売の高マージンと、装置産業であるボトリングの低マージンの非対称性にある。原材料費と製造コストが低い濃縮液は70%超の粗利を持つ。一方ボトラーは設備投資・燃料・人件費が重く、マージンは薄い。ゴイズエタはこの構造を「濃縮液ビジネスだけに集中すれば、資本効率が最大化される」と見抜いた。

### 構造図

```
[コカ・コーラ本社]
    保持: 秘密のコンセントレート製法 + 「コカ・コーラ」ブランド
    収入: コンセントレート販売（高マージン）+ ブランド使用料
    支出: 全国広告費 + R&D
         ↓ コンセントレート販売（独占供給）
[ボトラー（充填業者）—独占テリトリー制]
    保持: 工場・配送網・地域顧客関係・冷蔵設備
    収入: 完成品コーラの販売
    支出: コンセントレート仕入れ + 製造 + 配送 + 地域広告
         ↓ 完成品コーラを卸売
[流通チャネル: スーパー/コンビニ/レストラン/自販機]
         ↓ 小売販売
[最終消費者]
    ↑ 「コカ・コーラ」ブランドを認識・指名購入

ボトラーが失敗しても: コカ・コーラのブランドと製法は失われない
コカ・コーラが失敗しても: ボトラーのインフラは残る（が価値がなくなる）
→ 相互依存だが、コカ・コーラの方が交渉力が高い（製法の独占）
```

## 転換点の意味

### TP1: 1899年の「1ドル契約」——フランチャイズの誕生

ベンジャミン・トーマスとジョセフ・ホワイトヘッドがアサ・キャンドラーに提案した時、キャンドラーはほぼ断る気でいた。しかし彼らの熱意に押され、「まあ1ドル払ってくれるなら」と全米ボトリング権を永久に譲渡した。

なぜ転換点か: この契約によりコカ・コーラは「製法とブランドだけを持つ会社」という特異な構造を持つことになった。フランチャイジー（ボトラー）が地域インフラに投資し、フランチャイザー（コカ・コーラ）がブランドと原料供給を独占する——この構造はマクドナルド（1954年〜）や後のホテルフランチャイズより半世紀早い。コカ・コーラが世界展開できたのは、資本をほぼ使わずにローカルオペレーターの投資を引き出せたからだ。

### TP2: 1920年代の原液価格訴訟と「契約の限界」——ゆるい契約の代償

第一次大戦中、砂糖価格は3倍以上に高騰した。1899年の固定価格契約では、コカ・コーラはシロップを原価割れで売り続けるか、契約を破るかの二択に追い込まれた。コカ・コーラは値上げを試みたが、ボトラーたちは集団で提訴した。「コカ・コーラ・ボトリング対コカ・コーラ事件」は1920年代の業界最大の訴訟になった。

なぜ転換点か: フランチャイズ契約の「初期の寛大さ」が成長後に深刻な摩擦を生むという構造的問題を露呈した。この訴訟の一部和解（コカ・コーラはある程度の価格調整権を獲得）は不完全な解決に終わり、契約問題は1980年の「マスターボトラー契約」まで尾を引いた。「急成長のためにゆるい初期契約を結ぶと、後の再設計コストが膨大になる」という教訓として産業史に残る。

### TP3: 1986年コカ・コーラ・エンタープライズ設立——「純粋濃縮液カンパニー」への変身

ロベルト・ゴイズエタCEOは1986年、経営難に陥った中小ボトラー群を集約して「コカ・コーラ・エンタープライズ（CCE）」を設立し、49%を保持しながら株式市場に上場させた。この「再フランチャイズ化」により、コカ・コーラのバランスシートからボトリング設備の大半が消えた。

なぜ転換点か: ゴイズエタはコカ・コーラの事業モデルを根本から再定義した——「コカ・コーラは濃縮液カンパニーであり、装置産業ではない」。CCE上場後のコカ・コーラの投下資本利益率（ROIC）は急上昇し、ウォーレン・バフェットが1988年に6億ドルを投資した根拠もここにある。ブランドと製法だけを保持するビジネスモデルが「正しい」と証明された瞬間だった。

## 反対と執念

1899年の「1ドル契約」が何十年後に問題になるとキャンドラーは思っていなかった。彼の判断は当時の合理的な判断——「瓶入り飲料は都市のソーダファウンテンには勝てない」——に基づいていた。後にコカ・コーラを買収したロバート・ウッドラフ（1923年〜1954年社長）は、この契約問題と格闘しながら、一方でブランドへの投資を大幅に増やした。「どこでも飲める」「いつでも冷たい」という体験の一貫性を作るために、ウッドラフは戦時中の米軍への無料提供まで行った。

ゴイズエタの「再フランチャイズ化」は社内にも反発を呼んだ。「ボトラーを手放すと品質管理ができなくなる」「地域との関係が壊れる」という声があったが、ゴイズエタは「49%保持＋マスターボトラー契約での品質基準の明文化」で反論した。CCE上場後の株価上昇が批判を黙らせた。

## 考察・私見

コカ・コーラのボトラー制度が示す最も深い洞察は「何を外部化し、何を内部化するか」という判断の非対称性にある。製造・配送・地域営業は「ローカル知識と資本集約」を要求するが、価値の源泉にはならない。ブランドと製法は「非合理的な信念」によって守られ、複製できない。コカ・コーラはこの非対称性を直感的に（最初は偶然に）設計した。

TDL（東京ディズニーランド）の設計と比較すると面白い。TDLはIPをDisneyが保持し、OLC（オリエンタルランド）がオペレーションを持つ。コカ・コーラは製法をコカ・コーラが保持し、ボトラーがオペレーションを持つ。「IPないしブランド × フランチャイズオペレーション」の分離という意味では全く同じ構造だ。TDLがボトラー制度より50年遅れて同じ答えに到達したことは、このモデルの普遍性を示している。

ゴイズエタの1986年の改革は、後のARM（IPのみ保持、製造はTSMC）やSaaS企業（ソフトウェアのみ、インフラはAWS）という「最軽量資産・最高利益率」モデルの先駆けとも言える。「装置産業から知識産業へ」の転換は、コカ・コーラが最も早く大規模にやってみせた実験だった。

## 未解決の問い

- 1899年の契約交渉でキャンドラーはなぜ$1を受け取ったのか？完全無償でもよかったはずで、$1は法的効力（約因）のための象徴的対価だったのか
- マスターボトラー契約（1980年）の価格式の具体的な内容——CPIに連動させるだけでなく、どんな条件が埋め込まれたのか
- CCE設立後、コカ・コーラが「49%保持」を選んだのはなぜか？完全分離（0%）ではなく、品質管理と財務的な理由のバランスは？
- ペプシコのボトリングシステムはコカ・コーラと何が違うのか。ペプシが一時的に垂直統合（ボトラー完全子会社化）に振れた理由は？
- Netflixやクラウド企業の「サーバーはAWSに委託、コンテンツだけ保持」という構造は、コカ・コーラのボトラー制度の精神的後継か
