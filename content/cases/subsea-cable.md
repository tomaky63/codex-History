# 海底ケーブルの支配構造——国家から通信キャリア、そしてBig Techへ

## 俯瞰

「インターネットは無線で動いている」という誤解は根強い。実際には現代の国際インターネットトラフィックの99%は、太平洋・大西洋・インド洋の海底に敷設された光ファイバーケーブルを経由している。衛星通信が担う割合はわずか1%に過ぎない。

そのケーブルを「誰が所有するか」は、20世紀後半の60年間で3度、根本的に変わった。

**第1期（1956〜1988）：国家独占・コンソーシアムモデル**
1956年開通のTAT-1（世界初の大西洋横断電話ケーブル）から始まる第1期は、AT&T・BT・KDD（現KDDI）といった各国の国営または公益通信事業者が共同出資でケーブルを建設・運営するモデルだった。費用と収益を出資比率で分け合う「公益インフラ」の論理で動いていた。

**第2期（1988〜2001）：光ファイバー革命と過剰投資・崩壊**
1988年開通のTAT-8（世界初の大西洋横断光ファイバーケーブル）が容量を桁違いに引き上げた。インターネット商用化と重なり、1990年代のドットコムバブルで通信キャリアが競って大西洋・太平洋のケーブルを敷設した。しかし2001年のバブル崩壊で容量の96%が未使用状態に陥り、多くの事業者が破綻した。

**第3期（2008〜現在）：プラットフォーム企業による私有インフラ化**
Googleが2008年にアジア太平洋ケーブルへの投資を始めたのを皮切りに、Google・Meta・Amazon・Microsoftが次々と自社専用の海底ケーブルを建設・取得し始めた。2020年代には「Big Tech」が全太平洋・大西洋ケーブル容量の30%超を所有または専用利用している。

東京ディズニーランドがIPと運営の分離でOLCとDisneyの双方に利益をもたらしたのとは対照的に、海底ケーブルの第3期変革は「プラットフォームが物理インフラを取り込む垂直統合」——情報サービスと物理ケーブルを同一企業が支配する構造——が完成した事例だ。

## 構造分析

### 何が起きていたか

**第1期の構造（1956〜1988）**
AT&T・BT・KDD等の国営キャリアが共同出資でケーブルを建設・運営した。国際通話料金は「精算レート（settlement rate）」制度で各国の規制当局が管理し、ケーブル収益は出資比率で按分された。電話1通話あたりの収入モデルで成立するビジネスであり、「インフラは公益財」の論理が支配した。日本のKDDは、この枠組みの中でアジア太平洋の重要ケーブルに参加し続けた。

**第2期の転換（1988〜2001）**
TAT-8以降の光ファイバーケーブルは容量を指数関数的に拡大した。TAT-1が36回線だったのに対し、TAT-8は40,000回線以上を提供した。1990年代の電気通信自由化（米国1996年通信法など）が民間資本の参入を解禁し、1996年には「FLAG（Fiber-optic Link Around the Globe）」が初の純民間資本ケーブルとして完成した。しかし1999〜2001年の過剰投資とバブル崩壊で、建設済みケーブルの96%が「未使用暗ファイバー（dark fiber）」として放置された。

**第3期への転換（2008〜2020）**
Googleは2008年にアジア太平洋の「Unity」ケーブルコンソーシアムに参加。当初はキャリアとの共同出資者として振る舞ったが、2012年以降から完全自社所有ケーブルへシフトした。2016年にFacebook・MicrosoftがMARE NOSTRUM（大西洋）、Googleが単独所有のCurie（米国〜南米）を発表。2020年代には太平洋ルートの大半にBig Techが自社ケーブルまたは専用容量を確保している。

### なぜそうなったか（力学）

**第1の力：使用量の非対称性**
Google（検索・YouTube）・Meta（動画・画像）・Netflix・Amazonが生み出すトラフィックは、全インターネットトラフィックの60〜70%を占める。彼らは「最大の利用者」でありながら「ケーブルを持たない」という非対称状態にあった。帯域幅コストが収益に対して無視できない規模になれば、「借りるより持つ方が安い」という計算が成立する。

**第2の力：ドットコム崩壊後の資産価格崩壊**
2001年崩壊後、海底ケーブル資産の市場価格は建設コストの5〜10%まで下落した。数十億ドルで建設されたケーブルが数百億円で市場に出回り、財務体力のあるGoogleのような高収益IT企業には絶好の取得機会だった。崩壊が次世代の所有構造の礎を作った。

**第3の力：垂直統合の競争優位**
クラウドサービス（AWS・Google Cloud・Azure）の競争は「同じ性能をいかに低コストで提供するか」で決まる。データセンター間のトラフィックをゼロ限界費用で流せる自社ケーブルは、他社にはない構造的コスト優位になる。ケーブル内製化はサービス競争力に直結した。

**第4の力：地政学的フィルタリング**
9.11以降、米国政府はケーブルのルーティングと着陸権を安全保障の観点で管理し始めた。2019〜2020年には米国・日本・オーストラリアが中国系ケーブル会社（HMN Technologies、旧ファーウェイ海洋）の参加を相次いで制限。「信頼できる事業者」としてUS Big Techへの誘導が強まり、集中が加速した。

### 構造図

```
[第1期 1956-1988：国家コンソーシアム]
AT&T ──┐
BT  ───┼── 共同建設・共同運営 ──→ 銅線/同軸ケーブル ──→ 通話ごとの精算料金
KDD ───┘         (公益インフラ、出資比率で収益按分)

[第2期 1988-2001：光ファイバー+民間参入+崩壊]
既存キャリア + FLAG等の民間参入 ──→ 光ファイバーケーブル ──→ 過剰投資

Google ──┐
Meta   ──┼── 自社専用ケーブル ──→ 自社DCを直結 ──→ 自社サービスのコスト優位
Amazon ──┘

```

## 転換点の意味

### TP1: 1988年 TAT-8開通——容量の桁違いの変化（準備段階）

TAT-8が持つ意味は「高速化」ではなく「桁違いの容量化」だった。36回線→40,000回線という変化は、ケーブルを「希少なインフラ」から「大量の帯域を安価に供給できる資産」に転換させた。これが「通信は公益財」の論理を崩す出発点だ。インターネット商用化（1991年）とのタイミングの重なりが、1990年代の爆発的需要予測を生み、過剰投資ブームの土台を作った。

### TP2: 2001年 ドットコム崩壊——「次の秩序の礎」（逆説的転換点）

バブル崩壊は破壊だったが、同時に「資産の民主化」を起こした。建設コストの5〜10%で市場に出回った海底ケーブル資産は、財務的に生き残った企業に安値取得の機会を与えた。崩壊後の数年間で、Googleを含む次世代の覇者が格安でインフラ資産を積み上げた。この時期なしにBig Techの私有化は速度を持てなかった。逆説的に「崩壊がのちの独占の礎を作った」という点で、containerizationのMalcom McLeanがbreak-bulk港湾の崩壊を見てコンテナに全財産を賭けた構造と似ている。

### TP3: 2016年 Google・FacebookによるFOC的宣言（構造転換の完成）

2016年に複数のBig TechがキャリアなしのSelf-owned（完全自社所有）ケーブルを宣言したことは、単なる投資発表ではなかった。「情報サービス企業がインフラ層まで垂直統合する」という産業秩序の転換宣言だった。それまでの「インフラはキャリアが持ち、ITサービス企業が使う」という役割分担が終わった瞬間だ。

## 反対と執念

**伝統的通信キャリアの生き残り戦略**

AT&T・BT・NTT・KDDIは、Big Techによる私有化の波に対して真正面からの対抗ではなく、「分業再定義」で生き残りを図った。物理ケーブルの所有権をBig Techに譲りつつ、ケーブルの設計・施工・維持管理という高度な技術サービスに特化する道だ。

KDDIは日本国内の複数の「着陸点（cable landing station）」を保有し続け、日本への接続を必要とするBig Techのケーブルがここを経由せざるを得ない地理的ゲートキーパーの地位を確保した。「所有から管理へ」「インフラから着陸点へ」というポジションシフトだ。

**各国政府の安全保障的反発**

2019年、米国・日本・オーストラリア政府は太平洋ケーブルプロジェクト「Pacific Light Cable Network（PLCN）」への中国系企業参加を相次ぎ認可拒否した。これはケーブルへの盗聴装置埋め込みリスクを根拠とするが、本質は「情報インフラの支配権を誰が持つか」という純粋な地政学問題だ。

EU欧州議会は2023年に「欧州の海底ケーブルへの依存と脆弱性」に関する報告書を採択し、Big Tech私有化への規制的懸念を表明した。しかし市場の力に対する制度的対抗手段は現時点では限定的だ。

**研究者・市民社会の警告——「見えないインフラ」の問題**

ITU（国際電気通信連合）は1990年代からケーブル所有の集中リスクを繰り返し報告しているが、政策的対応は常に遅れた。理由の一つは「海底ケーブルが見えない」こと——地図に載らず、市民の目に触れず、問題が顕在化しにくい。抵抗が生まれにくいインフラの政治学だ。

## 考察・私見

海底ケーブルの歴史が示す最も重要な洞察は、**「最大の利用者は遅かれ早かれ所有者になる」** というメカニズムだ。

トラフィックの多数派を占める企業がそのトラフィックを運ぶインフラを内製化するのは経済的に自明だ。しかし「通信インフラは公益財」という20世紀の常識が、「通信インフラは競争優位の源泉」という21世紀の現実に塗り替えられる過程で、制度設計が完全に遅れた。

**コンテナ革命との対比**: McLeanのコンテナ革命では、標準規格が産業全体の効率を引き上げ、最終的に巨大コンテナ会社への集中を生んだ。海底ケーブルでも「IP over fiber」という共通標準の上で、最大の利用者が所有者になる集中が起きた。標準化が先にあり、集中が後から来る——物理インフラの共通パターンかもしれない。

**日本海運のFOC化との対比**: 日本海運では「国内事業者が国際競争圧力でコスト構造を転換」した。海底ケーブルでは「国際通信事業者が技術変化と資本力の差で新興勢力に取って代わられた」。どちらも従来の国家的な担い手が中核から外れていくが、FOC化は「逃げる話」、ケーブルの私有化は「奪う話」だ。

**日本のポジション**: 日本はアジア太平洋のケーブル交差点として地政学的に重要で、KDDIは着陸点という地位を保っている。しかしケーブル所有の主導権はすでにBig Techに移行している。「インフラを持つ者と、サービスを持つ者が分離する」構造は、日本の通信産業にとって既に現実だ。どう「着陸点の地政学的価値」を活かすかが次の問いだ。
