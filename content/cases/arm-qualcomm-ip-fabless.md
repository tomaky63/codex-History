# ARM/QualcommとIP・ファブレス型モデル——製造を持たない設計会社が半導体産業を支配した

## 俯瞰（Overview）

半導体産業の常識は長らく「垂直統合」だった。Intelはプロセッサを設計し、自社の工場（fab）で製造し、自社ブランドで販売する。この「一社で全部やる」モデルが1990年代まで支配的だった。

英国の小さなコンピュータ会社アコーン・コンピュータで1985年に生まれたARMプロセッサは、この常識を根底から覆す旅の始まりだった。1990年にApple・VLSI Technologyとの合弁でARM Holdingsが独立したとき、その会社のビジネスモデルは奇妙なものだった——チップを一枚も作らない半導体会社、だ。ARMは「命令セットアーキテクチャ（ISA）の設計図」をライセンスする。Apple・Qualcomm・Samsung・MediaTekなど数百社がARMのライセンスを取得して独自チップを設計し、それをTSMCやSamsungの工場で製造する。ARMはその連鎖の中で1チップ当たり数十円の小さなロイヤリティを受け取り続ける。

Qualcommは同じ1985年創業だが、異なる経路で「製造を持たない支配」に到達した。CDMA移動通信技術を業界標準として確立し、その標準必須特許（SEP）を持つことで、世界中の携帯電話メーカーから端末価格の3〜5%のロイヤリティを徴収するモデルを作り上げた。2000年に製造部門をEricssonに売却して完全ファブレスとなり、「設計と特許で世界最大の通信半導体企業」になった。

2007年のiPhone登場は、ARMとQualcommが組み合わさったエコシステムが40億台のスマートフォン市場を支える構造を確定させた。2020年のApple M1は「x86からARMへ」という半導体の最終的な覇権交代を告げた。

この事例の核心は「知的財産が技術スタックの最高価値層になる」という構造転換だ。製造のコモディティ化が進むほど、設計・アーキテクチャ・標準という「目に見えない資産」の価値が高まる。

## 構造分析

### 何が起きていたか

**1985〜1989年：ARMの誕生——「シンプルさ」という設計哲学**

1985年、英国ケンブリッジのアコーン・コンピュータ（BBC Micro製造元）でエンジニアのソフィー・ウィルソンとスティーブ・ファーバーが、新しいプロセッサ「Acorn RISC Machine（ARM）」を設計した。

設計の出発点は哲学的なまでにシンプルだった：「命令セットを極限まで削ぎ落とす（RISC：Reduced Instruction Set Computer）」。競合のMotorolaやIntelが「複雑な命令を1回で実行する（CISC）」方向に進んでいたのに対し、ARMは「シンプルな命令だけを高速で実行する」ことを選んだ。

このシンプルさの副産物が「低消費電力」だった。回路が単純なほどトランジスタ数が少なく、消費電力が下がる。当時のデスクトップPC時代には、電力は気にならなかった。しかし——10年後にバッテリー駆動のモバイル機器の時代が来たとき、このシンプルさが最大の武器になる。

**1990〜1999年：ARM Holdings設立——「純IP会社」という奇異なモデル**

アコーンがAppleのPDA「Newton」用のプロセッサを供給する商談が進む中、Apple・アコーン・VLSI Technology（半導体製造会社）の三者で、1990年にARM Holdings Ltd.を合弁設立した。

設立時の決断：「チップを製造しない」。

この決断は財務的な「弱さ」から生まれた。アコーンは製造設備を持てるほど大きくなかった。しかしスワンソンはこれを「強みに転換」する方法を考えた——製造しないことで、ARMは競合するチップメーカーの顧客にもなれる。Qualcomm・Samsung・Appleは互いにライバルだが、全員がARMのライセンスを取得できる。ARMは「半導体産業の中立的インフラ」として機能できる。

ARMのビジネスモデル（2つの収益源）：
1. **ライセンス料（License Fee）**：新しいIPコアのライセンス取得時に一括払い（数百万〜数億円）
2. **ロイヤリティ（Royalty）**：ARMアーキテクチャを使ったチップが出荷されるたびに1チップ当たり$0.05〜$0.20

1998年にロンドン証券取引所とNASDAQにIPOした時点で、ARMのIPコアは年間4,000万個以上のチップに搭載されていた。製造工場も在庫も持たないこの会社は、毎年確実に拡大するロイヤリティ収入を積み上げていた。

**1985〜2000年：Qualcommの軌跡——CDMA技術から特許帝国へ**

Qualcommは1985年、IrwinJacobsら7名の研究者がカリフォルニア大学サンディエゴ校（UCSD）から独立して設立した。初期事業は衛星通信（OmniTRACS）だったが、Jacobsは1989年にCDMA（符号分割多元接続）技術を移動体通信に適用する構想を発表した。

当時、移動体通信の主流はヨーロッパ主導のGSM（TDMA方式）だった。CDMAは理論的に周波数利用効率が高く、容量が大きいが、Qualcommの完全な特許支配下にあった。1994年、米国でCDMAが IS-95標準として採択され、その後Verizon・Sprintが採用。2001年に3GとなったW-CDMAでも、Qualcommの特許が必須特許（SEP）として組み込まれた。

Qualcommの特許ビジネスの論理：
- CDMA技術特許を業界標準に組み込む（FRAND：公正・合理的・非差別的条件でライセンスする義務）
- 全てのCDMA端末メーカーはQualcommにロイヤリティを支払う義務がある
- 2G・3G・4G（LTE）・5Gと標準が更新されるたびに、Qualcommの必須特許も更新された

2000年の戦略的転換：Qualcommは製造部門（IntegratedCircuit（IC）事業部とインフラ・システム事業部）をEricssonに売却し、「純粋な設計・特許会社」への転換を完了した。この決断はARMが1990年にした「製造しない」決断と同型だった——ただしQualcommは「チップ設計自体は続ける（ファブレス）」という点でARMとは異なる。

**2001〜2007年：ファブレスとFoundryの二人三脚**

2000年代前半、ファブレス設計会社とTSMC・Samsung等のFoundryが二人三脚で半導体産業を再編した。ARMがアーキテクチャを設計・ライセンスし、ファブレス各社（Qualcomm・NVIDIA・Broadcom）が独自チップを設計し、Foundryが製造する——この三層構造が確立した。

ARMアーキテクチャ採用の拡大（2000年代初頭）：
- Nokia・Motorola・SamsungのフィーチャーフォンにARMチップが標準採用
- iPod（2001年）にARM7が搭載
- Texas Instruments・Freescaleがモバイル向けSoCでARM採用
- ARMは毎年10億個以上の出荷ライセンス数を積み上げていった

**2007年：iPhone——ARMとQualcommの「本番」**

2007年1月9日のiPhone発表は、ARM×Qualcommエコシステムの存在意義を一挙に証明した。

初代iPhoneのチップ構成：
- アプリケーションプロセッサ：Samsung製・ARM11コア搭載（ARM Holdings ISAライセンス使用）
- 通信チップ（ベースバンド）：Infineon製だが、Qualcommのパテントライセンス使用
- GPSチップ：他社製だがARM使用

スマートフォンは「ARM命令セット」と「CDMA/LTE特許」の二つなしには成立しなかった。世界で年間10億台以上のスマートフォンが生産されるようになった2010年代、ARMとQualcommが収受するロイヤリティは天文学的な規模になった。

**2010〜2020年：Apple Siliconとx86覇権の終焉**

2010年、Appleは自社設計ARMチップ「A4」をiPadに搭載した。ARMのISAライセンスを取得して「自分でARMコアを設計する（ARM Custom ISA）」という最も高度なARMライセンス形態に移行した。

AppleのA-seriesチップの進化（ARM Custom ISA）：
- A4（2010）→ A7（2013：64ビット化、業界初）→ A12X（2018）→ M1（2020）

2020年11月、AppleはMacに「M1」チップを搭載した。IntelのCore i7を性能・電力効率の両面で大幅に上回るM1は、「ARMはIntelに劣る」という最後の偏見を消し去った。20年間Intelへの依存を維持していたMacが、ARMアーキテクチャに全面移行することを宣言した。

x86（Intel）が支配してきたPC・サーバー市場で、ARMが本格的に競争できることが証明された瞬間だった。

### なぜそうなったか（力学）

**力学1：RISCの「シンプルさ」がモバイル時代のキラーアドバンテージになった**

1985年に「技術的な洗練さ」から選ばれたシンプルな命令セット設計が、20年後の「バッテリー駆動端末」という市場で最大の競争優位になった。

| 比較 | ARM（RISC） | Intel x86（CISC） |
|---|---|---|
| トランジスタ数 | 少ない（設計シンプル） | 多い（複雑な命令デコード回路） |
| 消費電力 | 1〜5W | 15〜100W |
| 発熱 | 低い | 高い（冷却が必要） |
| デスクトップPC | 不利（処理速度で劣った） | 優位 |
| スマートフォン | 圧倒的優位 | 実質不可能（熱・電池） |

Intelは2006〜2007年のスマートフォン勃興期に「モバイル向けAtom」プロセッサを開発したが、ARMとの電力効率の差を縮めることができなかった。「最初の設計哲学の差」は30年経っても埋まらなかった。

**力学2：ARMのネットワーク効果——生態系が生態系を呼ぶ**

ARMのライセンスモデルは、採用者が増えるほど強くなるネットワーク効果を持つ。

- ARMを採用するチップメーカーが増える → ARMアーキテクチャ向けコンパイラ・開発ツール・OSが充実する
- 開発ツールが充実する → 新しいチップメーカーがARMを採用しやすくなる
- 新規採用者が増える → ARMの収益が増え、次世代アーキテクチャへの投資が可能になる

この正のフィードバックループが回り始めると、後から別のISAでゼロから生態系を作ることは実質不可能になる。RISC-V（2010年代に登場したオープンソースISA）がARMに挑んでいるが、20年以上蓄積されたツール・ソフトウェア・人材の生態系を短期間で追いつくことは困難だ。

**力学3：QualcommのSEP（標準必須特許）戦略——技術が標準になることで特許が永続化する**

通常の特許は「他社が回避できる代替技術を開発すれば無価値化する」リスクがある。しかしQualcommのCDMA/LTE/5G特許は「回避できない」——その技術自体が業界標準になっているから。

標準必須特許（SEP）の論理：
1. Qualcommが独自技術（CDMA）を開発する
2. 標準化団体（3GPP等）に提案し、業界標準として採択させる
3. 標準採択後は「FRAND条件でライセンスする義務」があるが、ライセンスを断ることはできない
4. 世界中で標準に準拠する全メーカーが支払い義務者になる

Qualcommはこの戦略を2G→3G→4G→5Gと反復した。標準が更新されるたびに、Qualcommの必須特許ポートフォリオも更新された。これは「特許の有効期限切れリスク」を技術更新で打ち消す永続的な収益構造だ。

**力学4：ファブレス化が製造コストのボラティリティをリスクヘッジした**

Intelのような垂直統合企業は、最先端製造技術に巨額投資（数兆円規模のfab建設）を続ける義務がある。技術が陳腐化すれば投資が無駄になる。

ARMとQualcommのファブレスモデルは「製造コストをゼロ変動費化」した。TSMCが製造プロセスを更新すれば、ARMとQualcommの設計は自動的に恩恵を受ける。製造への投資リスクはTSMC（Foundry）が引き受け、IP企業は設計に集中できる。

## 転換点の意味

### TP1: 1990年——ARM Holdings独立と「製造しない半導体会社」の誕生

Apple・Acorn・VLSIの三者合弁設立という形式は、それ自体が「一社で全部やらない」という新しい産業構造の宣言だった。

なぜ転換点か：半導体IPの「ライセンス専業会社」という前例がなかった。ARMは「設計の設計図をライセンスする」という何重にも抽象化されたビジネスモデルを採用した。Intelが「チップ」を売るなら、ARMは「チップを作るためのレシピ」を売る。この抽象化レベルの差が、後のスケールアウトを可能にした——ARMがライセンスを取得した会社が作ったチップの総出荷数は2020年代に年間250億個を超えた。

### TP2: 2000年——Qualcommの製造部門売却と完全ファブレス化

1999〜2000年のドットコムバブルの絶頂期に、Qualcommは逆張りで製造資産を手放した。「製造を持ち続けることは戦略的に正しくない」というJacobsの判断だった。

なぜ転換点か：QualcommはCDMA特許を持つ「標準の守護者」だった。製造部門を持ち続けることは、顧客（他のチップメーカー）との競合関係を生む。製造から撤退することで「Qualcommは競合ではなく、必要なライセンスを提供するパートナー」という立場が明確になった。ARMが「中立的IP会社」として成立した構図と同型の「垂直解体による中立化」戦略だった。

### TP3: 2007年——Apple iPhoneとARMエコシステムの大爆発

ARMがライセンスを開始してから17年、iPhoneは「ARMモデルは正しかった」ことを年10億台規模で証明した。

なぜ転換点か：それまで「組み込み系・フィーチャーフォン向け」のマイナーな技術だったARMが、「コンシューマーの最重要デバイス」の心臓部になった。スマートフォン→タブレット→スマートスピーカー→スマートウォッチ→EV車載コンピュータと連鎖する10億台規模の市場で、ARMが「目に見えないが全部に入っている標準インフラ」になった瞬間。1チップ当たりの単価は低くても、累積規模が利益を生む「ロングテール・ロイヤリティ」モデルの完成。

## 反対と執念

Qualcommの特許ライセンス戦略は、業界から常に強い批判を受けた。AppleはQualcommに対して2017年に特許使用料をめぐって訴訟を起こし（「クアルコムは競争に反するやり方で高額な料金を課している」）、全世界で70件以上の訴訟が同時進行した。FTC（米連邦取引委員会）もQualcommの慣行について反競争的行為として調査・提訴した。2019年にAppleとQualcommは$45億の和解に達し、Qualcommは引き続きAppleにモデムチップを供給することで合意した——つまりAppleも「Qualcommなしでは5Gができない」という現実を認めた。

ARMのIPOと成長は「英国は世界的な技術企業を育てられるか」という議論の象徴でもあった。2016年にSoftBankが£243億（$314億）でARMを買収したとき、英国内から「重要な技術資産を日本企業に売り渡した」という批判が出た。その後2020年にNVIDIAが$400億での買収を発表したが、EU・英国・米国の規制当局に独占懸念で2022年に阻止された。2023年、ARMはNASDAQに単独上場（時価総額$600億超）した。

## 考察・私見

ARMのモデルで最も深い洞察は「中立性の経済的価値」だ。ARMはQualcomm・Apple・Samsungが互いに激しく競合しているにも関わらず、全員にライセンスを販売できる。これが可能なのは、ARMが「誰のチップも製造しない」からだ。製造を持つと、どのメーカーの注文を優先するかで利益相反が生まれる。「何もしない」ことが最大の武器になる逆説的なビジネスモデルだ。

Qualcommの特許戦略についての評価は難しい。「CDMA技術の発明・普及に真に貢献した発明者が、正当な対価として知的財産収益を得る」のは正当だ。しかし「FRAND義務があるにも関わらず、交渉力を利用して過大なロイヤリティを課す」のは独占的慣行だという批判も正しい。知的財産法の設計が「発明の保護」と「競争の維持」のどちらを優先するかによって、同じ行動が合法にも違法にもなりうるという根本的な曖昧さをQualcommは体現している。

2020年代のリスクは2方向から来ている。一つはRISC-V（オープンソースISA）の台頭——中国がARMへの依存を削減するため国家支援でRISC-Vエコシステムを構築しようとしている。もう一つは「AIチップの主戦場がARMでもx86でもなく、GPGPUやNPU（ニューラル処理ユニット）になる」という可能性だ。NVIDIAが「AIの時代のIntel」になるかもしれない。
